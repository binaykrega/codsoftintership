# codsoftintership
project for codsoft internship in AI 
This project is a simple chatbot developed using Python. The main objective of the chatbot is to simulate a basic conversation with a user by responding to different types of text inputs. It works by identifying specific keywords in the user's message and generating an appropriate predefined response.
The chatbot can greet users, introduce itself, remember the user's name, answer simple questions, tell jokes, share interesting facts, display the current date and time, and respond to basic queries such as weather, news, and translations. If the chatbot does not recognize the user's input, it politely asks the user to rephrase the question.
